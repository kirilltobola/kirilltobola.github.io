CTA framework

Инструментарий для решения задачи семантической интерпретации столбцов таблицы (Column Table Annotation) с помощью LLM.

Инструмент предстваляет собой адаптацию идеи "развертки" таблицы в последовательность для русского языка, представленную в статье.

Инструмент разрабатывается при поддержке ИСП РАН и ИДСТУ СО РАН.


Используемые технологии:
  • Python
  • PyTorch
  • Transformers
  • BERT

CTA dataset

Русскоязычный датасет для решения задачи семантической интерпретации столбцов таблицы (Column Table Annotation).

За основу был взят датасет ru-wiki-tables - срез русскоязычных таблиц википедии за 2021 год.

Базовый датасет был очищен и размечен для обучения модели, решающей задачу семантической интерпретации столбцов таблицы.
